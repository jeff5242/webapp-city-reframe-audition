## 版本選擇規則

**以文件的「填表日期」（審議資料表）或「報核日期」（申請書）為準：**

```
填表日期 < 2019-01-30          → 107年版
2019-01-30 ≤ 填表日期 < 2022-03-24  → 108年版
2022-03-24 ≤ 填表日期 < 2024-12-03  → 111年版
填表日期 ≥ 2024-12-03          → 113年版
```

> 若無法解析填表日期，系統預設套用 111年版規則並標記警告。

## 各版本包含的文件

所有版本均包含：
- 附件1-1：事業概要計畫書範本
- 附件1-2：事業概要計畫書附件冊範本
- 附件2-1：事業計畫書範本
- 附件2-2：事業計畫書附件冊範本
- 附件3-1：權利變換計畫書範本
- 附件3-2：權利變換計畫書附件冊範本
- 修訂說明（各版本對照差異）

108年版以上額外包含：
- 附件4：自劃更新單元併送事業計畫(事業概要)簡化程序申請書
