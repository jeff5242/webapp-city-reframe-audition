# 臺北市都市更新事業計畫書 111年版 規章 Wiki

**版本**：111年3月24日修正公布版（Revised March 24, 2022）  
**適用法規**：都市更新條例 / 臺北市都市更新自治條例 / 臺北市都市更新建築容積獎勵辦法  
**版本沿革**：106年初版 → 107年3月修正（配合規劃設計獎勵）→ 108年全面修正（配合都更條例及相關子法）→ 111年全面修正（配合自治條例及實務執行檢討）

## 111年修訂重點（14項）

1. **申請書 / 切結書 / 委託書** — 新增公辦都更案注意事項提醒
2. **審議資料表** — 新增無障礙車位欄位、充電車位欄位；需載明外審程序（都設、都計、文資、捷運影響評估）
3. **更新單元範圍** — 若位於公劃更新地區且符合自治條例第12條第4款，需依「臺北市畸零地使用自治條例」載明相鄰土地「業已建築完成」說明
4. **辦理緣起 / 更新單元檢討書指導內容 / 附錄「更新單元核准函」** — 依自治條例第15條修正，新增範例及注意事項
5. **現況分析** — 無建物登記但有其他合法建築物證明者，需自行檢核有無登載；修正地上物拆遷計畫載明方式
6. **申請容積獎勵項目及額度** — 依都更條例第65條修正獎勵上限；新增臺北市都更建築容積獎勵項目檢核表；建築設計圖說移至「建築興建計畫」章節；簡化建築師簽證表格
7. **財務計畫** — 修正營業稅提列公式；新增都更規劃費用 / 估價費用計算公式；新增協議合建財務計畫格式；事業計畫含權利變換時，需附核定事業計畫「財務計畫」成本說明對照
8. **都市設計審議** — 「臺北市都市設計審議報告書」部分圖說章節納入計畫書注意事項（若需經都設審議者，請依「都設審議報告書」中「○○○製作」）
9. **人行步道面積檢討圖 / 一樓平面圖** — 新增標示公用設施突出物（燈桿、交通號誌桿、箱、消防栓、停車收費亭、停車計時器、郵筒、電話亭、變電箱、垃圾桶、座椅、候車亭等）；共專用圖說補充充電車位、無障礙車位
10. **產權登記清冊（權利變換計畫書）** — 擬請幹事機關提供意見後修正
11. **同意書注意事項** — 依109年「臺北市政府受理都市更新案審查與處理同意書重複出具及撤銷作業要點」新增注意事項；新增有關未填具完整之所有權人權利價值比率或分配比率及新同意書格式提醒
12. **公聽會登報方式（第32條）** — 新增「或新聞電子報」；附件新增「或新聞電子報證明」
13. **部分權利變換 / 部分協議合建案** — 審議資料表、財務計畫章節新增範例及修改注意事項
14. **事業概要** — 新增附錄須檢附自辦公聽會相關資料

## 計畫書製作規範

### 版面格式
- **紙張**：A3 (42×29.7 cm)，橫式雙面印刷
- **裝訂**：加封面裝訂於左側短邊；封面紙張 150 磅以上；封面勿插入圖片；勿以塑膠封膜或銅板紙類裝訂
- **列印**：標示以彩色列印；其餘得以黑白列印
- **蓋章**：申請書 / 切結書 / 委託書 / 審議資料表需附用印之正本；其餘版本得以影本替代（需加蓋「與正本相符章」）

### 數字格式
- 地段小段、道路名稱用國字；其他一律阿拉伯數字、半形
- 千分位分隔符號：「,」（逗號）
- 表格數值靠右對齊
- 金額：四捨五入至整數位
- 面積 / 百分比欄位：至少取至小數第二位
- 單位：面積用 m²；公尺用 M

### 引用資料
- 統計計畫圖如引用他處，需註明「資料來源」，避免著作權問題

## 版本適用規則

| 情境 | 適用版本 |
|-----|---------|
| 新案申請 | 111年版（本版） |
| 變更都市更新事業計畫（以106年範本擬稿） | 106年版範本，附變更項目說明表，配合原核定版對照頁次說明 |
| 舊法變更案（適用新法） | 108年版（不須製作新舊對照），但附錄需含都更審議會會議紀錄；涉及原核定承諾事項仍需製作新舊對照 |
| 幹事會審議階段 | 沿用公展版（有蓋騎縫章），所有數字皆須可辨識 |

## 流程關鍵節點

```
自行劃定更新單元
    ↓ 更新單元核准函 (附錄二)
事業概要核准 (附錄三) ← 可選
    ↓
事業計畫書送件
    ↓ 幹事會審查（沿用公展版）
公開展覽 30 天 + 公聽會
    ↓
都更審議委員會審議
    ↓ [如需] 都設審議 / 捷運影響評估 / 文資審查
核定事業計畫
    ↓ [如採權利變換]
權利變換計畫書送件
    ↓
核定權利變換計畫
    ↓
施工 → 地政登記
```
